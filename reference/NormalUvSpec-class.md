# Univariate Gaussian component specification

Univariate Gaussian component specification

## Slots

- `name`:

  Fixed to `"normal-uv"`.

- `paramNames`:

  `c("mu", "s2")`.

- `priorSpec`:

  Filled by
  [`defaultPrior`](https://madsyair.github.io/nimix/reference/defaultPrior.md).
