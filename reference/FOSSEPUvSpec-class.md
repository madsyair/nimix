# FOSSEP mixture components (Fernandez-Steel skew exponential power)

Four-parameter neo-normal component with location `mu`, scale `sigma`,
skewness `alpha` (`alpha = 1` symmetric) and shape `theta` (`theta = 2`
skew-Normal kernel). Non-conjugate.

## Usage

``` r
FOSSEPUvSpec()
```

## References

Choir, A. S. (2020). The New Neo-Normal Distributions and their
Properties. Dissertation. Institut Teknologi Sepuluh Nopember.
