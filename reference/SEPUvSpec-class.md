# SEP mixture components (symmetric exponential power)

Symmetric exponential-power (generalised error) component family with
location `mu`, scale `sigma`, and shape `nu` governing tail weight
(`nu = 2` Normal, `nu = 1` Laplace). Non-conjugate; NIMBLE assigns
adaptive samplers to the component parameters.

## Usage

``` r
SEPUvSpec()
```

## References

Choir, A. S. (2020). The New Neo-Normal Distributions and their
Properties. Dissertation. Institut Teknologi Sepuluh Nopember.
