# FSSN mixture components (Fernandez-Steel skew Normal)

Fernandez-Steel skew-Normal component with location `mu`, scale `sigma`,
and skewness `alpha` (`alpha = 1` Normal). A log-normal prior on `alpha`
treats left/right skew symmetrically. Non-conjugate.

## Usage

``` r
FSSNUvSpec()
```

## References

Fernandez, C. & Steel, M. F. J. (1998). On Bayesian modeling of fat
tails and skewness. JASA 93, 359–371. Choir, A. S. (2020). The New
Neo-Normal Distributions and their Properties.
