# Neighbours of one region

Neighbours of one region

## Usage

``` r
neighborsOf(spec, region)

# S4 method for class 'SpatialWeightSpec'
neighborsOf(spec, region)
```

## Arguments

- spec:

  A
  [`SpatialWeightSpec`](https://madsyair.github.io/nimix/reference/SpatialWeightSpec-class.md).

- region:

  A region id (character) or index (integer).

## Value

Character vector of neighbouring region ids.
