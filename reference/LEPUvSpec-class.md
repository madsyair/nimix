# LEP mixture components (exponential power, alternative parameterisation)

Symmetric exponential-power component with location `mu`, scale `sigma`,
and shape `nu`; an alternative parameterisation to
[`SEPUvSpec`](https://madsyair.github.io/nimix/reference/SEPUvSpec-class.md).
Non-conjugate.

## Usage

``` r
LEPUvSpec()
```

## References

Choir, A. S. (2020). The New Neo-Normal Distributions and their
Properties. Dissertation. Institut Teknologi Sepuluh Nopember.
